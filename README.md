# Project Complexity Intelligence Engine

A premium hackathon-ready expert system that analyzes software projects and generates:
- complexity score
- difficulty level
- explainable rule trace
- budget signal
- weekly roadmap
- scope reduction plan
- what-if simulation
- project comparison
- browser history of analyses
- AI copilot answers

## Run locally

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm run dev
```

Frontend runs on `http://localhost:5173`
Backend runs on `http://localhost:5000`

## Main features
- **Why This Score?** panel with triggered rules
- **MVP vs Full Product** analysis mode
- **What-if simulator** for team size, timeline, integrations, and modules
- **Project comparison** page
- **Suggested scope reduction** panel
- **Weekly roadmap** output
- **Budget estimation** card
- **Domain-specific rules** for AI, mobile, e-commerce, IoT, and management systems
- **Local history** of recent analyses
- **AI copilot** endpoint for quick expert answers
